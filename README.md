🚀 YOLO Object Detection Project
📌 Overview

This project implements real-time object detection using the YOLO (You Only Look Once) algorithm. The model detects multiple objects in images, videos, or live webcam feeds with high accuracy and speed.

The goal of this project is to understand and apply deep learning concepts in computer vision while building a practical application.

🧠 What is YOLO?

YOLO (You Only Look Once) is a state-of-the-art, real-time object detection system that:

Detects objects in a single pass
Divides the image into grids
Predicts bounding boxes and class probabilities simultaneously
✨ Features
🔍 Real-time object detection
🎥 Works on images, videos, and webcam
⚡ Fast and efficient detection
🧠 Pre-trained model support (COCO dataset)
📦 Easy to extend for custom datasets
